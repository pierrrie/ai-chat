# Админка draxter.aichat (v2.1)

**Настройки → Настройки модулей → draxter.aichat**

## Вкладки

| Вкладка | Назначение |
|---------|------------|
| Виджет | Включение, приветствие, аватар, **кнопка FAB** (цвет, текст, позиция), автоподключение |
| AI | Провайдер (Gemini / DeepSeek), ключи, SSL |
| Промпты | Системные инструкции, справочник FAQ, **превью FAQ** |
| Намерения | Regex для small talk, покупки, подбора товаров |
| Голос | STT (Gemini), **TTS** (browser / Yandex), лимиты |
| Каталог | YML / файл / инфоблок, кэш |
| CRM | Bitrix24, CRM сайта, email, маппинг полей, Метрика |
| Статистика | Сессии, сообщения, ошибки, лиды за 7/30 дней |
| Логи | api-*.log, просмотр JSON-сессии, очистка старых логов |

## Голос (TTS)

- **browser** — сервер возвращает JSON, озвучка через `speechSynthesis` (без внешних API).
- **yandex** — Yandex SpeechKit: API Key + Folder ID + голос (`jane` и др.).

Включите **Отвечать голосом** и выберите провайдер на вкладке «Голос».

## Превью FAQ

На вкладке «Промпты»: введите тестовый вопрос → **Проверить FAQ**. Показываются подобранные секции и фрагмент, попадающий в промпт (без полного ответа LLM).

## Статистика и логи

- **Статистика** строится из JSON в `/upload/draxter_aichat_logs/` (или `CHAT_LOG_DIR`).
- **Логи API** — файлы `api-YYYY-MM-DD.log` (Gemini, DeepSeek, Bitrix24, email и т.д.).
- Ссылка «лог» у диалога открывает расшифровку сессии на вкладке «Логи».

## Health

`/local/ajax/draxter_aichat.php?action=health` — проверка ключей, каталога, голоса, CRM.
