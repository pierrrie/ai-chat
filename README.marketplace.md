# Публикация модуля draxter.aichat в 1С-Битрикс.Маркет

## Требования к серверу

- PHP 7.4+ (рекомендуется 8.1+)
- Расширения: `curl`, `mbstring`, `json`
- 1С-Битрикс: Управление сайтом / Интернет-магазин
- Для каталога из инфоблока: модуль `iblock`

## Внешние сервисы (указать в карточке решения)

| Сервис | Назначение | Ключ |
|--------|------------|------|
| Google Gemini / DeepSeek | Текстовые ответы | Настройки модуля → вкладка AI |
| Yandex SpeechKit | TTS (опционально) | `YANDEX_TTS_*` |
| Google Gemini multimodal | STT (опционально) | `GEMINI_API_KEY` |
| Bitrix24 REST | Лиды (опционально) | Webhook URL |
| Браузер speechSynthesis | TTS без API (по умолчанию) | — |

В карточке маркетплейса отметьте: **«Пользователю необходимо оплачивать внешний сервис»**.

## Сборка дистрибутива

1. Не включайте в архив файл `config/aichat.config.php` с реальными ключами.
2. Включите: `local/modules/draxter.aichat/`, `local/components/draxter/`, `local/ajax/draxter_aichat.php`.
3. Версия модуля: **2.1.0** (`install/version.php`).
4. Документация: `docs/INSTALL.ru.md`, `docs/ADMIN.ru.md`, `docs/CRM.ru.md`.

## Установка у покупателя

1. Установить модуль `draxter.aichat` через «Маркетплейс → Установленные решения».
2. **Настройки → Настройки модулей → draxter.aichat** — заполнить API, каталог, промпты, виджет.
3. Подключить виджет:
   - В footer шаблона: `<?$APPLICATION->IncludeComponent('draxter:aichat.chat', '', []);?>`
   - Или включить «Автоподключение на всех страницах» в настройках.

## Проверка перед модерацией

- [ ] `/local/ajax/draxter_aichat.php?action=health` — JSON с `hasApiKey: true`
- [ ] Чат отвечает по каталогу
- [ ] Голос: запись → текст → ответ (TTS: browser / Yandex)
- [ ] CRM: лид с `page_url` в комментарии; маппинг UF (опционально)
- [ ] Админка: вкладки Статистика, Логи, превью FAQ
- [ ] Мультисайт: разные приветствия для разных `site_id` в настройках
- [ ] Сканер безопасности / монитор качества партнёрского кабинета

## Тестовые данные для модератора

В карточке решения укажите тестовый `GEMINI_API_KEY` и URL демо-сайта с установленным модулем.
